# heychat
heychat is a CLI GPT wrapper on Linux (soon maybe on Windows/Mac) for ChatGPT (soon maybe on other LLMS)

# API Key Storage
Set API key with `.\heychat --key=$OPEN_AI_API_KEY` \
API key is stored in /home/$USER/.heychat
